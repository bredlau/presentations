Statische Code-Analyse für Groovy & Grails mit CodeNarc=======================================================Dieser Vortrag stellt CodeNarc - ein Werkzeug für die statische Code-Analyse für Groovy & Grails - vor. Mit CodeNarc kann Groovy-Code auf Defekte, schlechte Praktiken, Inkonsistenzen und Formatfehler überprüft werden. CodeNarc stellt außerdem ein flexibles Framework für Regeln und Regelwerke bereit. Über Plugins kann CodeNarc in das Build-Management über Ant, Maven, Gradle, Grails, Griffon, Sonar und Hudson eingebunden werden. Eine Live-Demo demonstriert den Einsatz von CodeNarc.
